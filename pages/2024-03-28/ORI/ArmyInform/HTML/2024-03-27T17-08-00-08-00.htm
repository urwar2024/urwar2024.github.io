Кількість постраждалих внаслідок російського удару 27 березня зросла до 16 людей.

Про це повідомив голова Харківської облдержадміністрації Олег Синєгубов.

«За попередніми даними, сьогодні ворог вперше по Харкову завдав удару саме керованим боєприпасом великого калібру», — написав Олег Синєгубов.

Застосування авіації для удару по Харкову підтвердив і керівник Слідчого управління поліції Харківщини Сергій Болвінов.

«По Харкову вдарили авіабомбами — вперше з 2022-го року.  Це УМПБ Д-30 СН, що виготовлений на базі ФАБ-250. Маємо два прильоти — перше влучання по загальноосвітній школі, друге прямо посеред житлових будинків«, — написав Сергій Болвінов.

Він зауважив, що авіацію для удару по житлових кварталах Харкова росіяни не використовували вже два роки.

Пізніше голова Харківської ОВА Олег Синєгубов повідомив, що для удару по Харкову росіяни використали боєприпас УМПБ Д-30, який за силою вибуху — середнє між авіабомбою та ракетою, застосовується, як і з літака, так і з РСЗВ «Смерч».

Як повідомляла раніше АрміяInform, унаслідок удару по Харкову одна людина загинула.
